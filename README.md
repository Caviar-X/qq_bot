## 使用方法

运行

```bash
cargo build --release
```

之后在`target/release`里找到qq_bot可执行文件

将qq_bot放到你喜欢的地方，运行

```bash
chmod +x qq_bot #提权，若有此权限不用执行
./qq_bot --uin QQ号 --password 密码
```

第一次运行需拿手机扫码
